# SSHR

- SSHR là một ứng dụng TUI (Text User Interface) để quản lý và kết nối với các máy chủ thông qua giao diện terminal.

## Get started
- `sshr` sẽ đọc file host từ ~/.ssh/config

## Cài đặt

```bash
git clone https://github.com/hoangneeee/sshr.git

cd sshr

cargo build --release
```
